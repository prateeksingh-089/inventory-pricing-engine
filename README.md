1-Web Dashboard Development: Built an interactive web app using Python and Streamlit to help store managers track their stock levels and sales in real time.

2-Smart Pricing System: Programmed an automatic pricing feature using Pandas that raises prices by 25% when stock is running low, and drops prices by up to 50% for items close to their expiration date to clear them out fast.

3-Sales & Waste Tracking: Created data tracking tools to automatically log every customer checkout, calculate daily profit margins, and track the financial loss from expired items.

4-Cloud Deployment: Set up the project on GitHub and deployed it live to the web using Streamlit Cloud so anyone can access and test the app via a public link.
